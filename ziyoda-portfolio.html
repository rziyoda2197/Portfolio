Men haqimda

O'zbekiston, Xorazm viloyatida tug'ilib o'sdim. "Al-Xorazmiy vorislari" loyihasida ta'lim olib, dasturlash sohasida puxta bilim va ko'nikmalarga ega bo'ldim.

Hozirda 3 yillik amaliy tajribaga ega Full-Stack dasturchi sifatida ishlayapman. Backend tomonida Python, Django, Flask va Django REST Framework (DRF) texnologiyalaridan foydalanaman.

Frontend sohasida HTML va CSS orqali zamonaviy, responsive interfeyslар yarataman. Ma'lumotlar bazasi sifatida SQLite va PostgreSQL bilan ishlash tajribam bor.

Mening maqsadim — texnologiyalarni insonlar hayotini osonlashtiradigan mahsulotlarga aylantirish. Har bir loyihada kod sifatiga va yaxshi arxitekturaga alohida e'tibor beraman.
